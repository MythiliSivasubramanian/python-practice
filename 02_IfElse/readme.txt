02_IfElse - Python practice scripts

This folder contains beginner Python programs that practice conditional logic
using `if`, `elif`, and `else` statements.

Files in this folder:

- calculator.py : Simple calculator for +, -, *, and / using user input.
- discount_calculator.py : Applies discount based on purchase amount and prints final amount.
- divisible_5_11.py : Checks whether a number is divisible by both 5 and 11.
- exam_score_grading.py : Validates exam score input and returns grade (A-F), with repeat option.
- leap.py : Checks whether a given year is a leap year.
- multiple_3_7.py : Checks whether a number is a multiple of 3 or 7.
- simple_login_check.py : Validates username and password against fixed credentials.
- temp_check.py : Classifies temperature as hot, cold, or normal.
- valid_triangle.py : Checks whether three sides can form a valid triangle.
- vowel_consonant.py : Checks whether an entered character is a vowel or consonant.
