01_Basics - Python practice scripts

This folder contains beginner-level Python programs that practice input, loops,
strings, lists, and basic functions.

Files in this folder:

- favourite_color.py : Takes user name and favourite color, then prints a greeting.
- duplicates_list.py : Finds duplicate values in a list.
- remove_duplicates.py : Removes duplicate items from a list while keeping order.
- squares:of_numbers.py : Returns the squares of numbers in a list.
- even_numbers_sum.py : Sums only the even numbers in a list.
- factorial.py : Calculates factorial of a number using a loop.
- count_word_frequency.py : Counts frequency of each word in a sentence.
- mutable_immutable.py : Demonstrates mutable vs immutable behavior using ids.
- deep_copy_shallow_copy.py : Demonstrates shared references, shallow copy, and deep copy.
- smallest_positive_number.py : Finds the smallest positive number in a list.
- sum_digits.py : Returns the sum of digits of a number.
- validate_email_format.py : Performs a simple email format check.
- count_vowels.py : Counts vowels in a given string.
- palindrome.py : Checks whether a string is a palindrome.
- vowels_consonants_count.py : Counts vowels and consonants in a string.
