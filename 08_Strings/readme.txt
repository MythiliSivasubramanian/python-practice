This folder has small examples and exercises all about working with strings in Python.

If you are new to Python, this is a great place to practice:

- Learn how to create and change text using quotes and special characters.
- See how to join (concatenate) strings or repeat them.
- Find out how to split sentences into words and pull out parts of text.
- Try simple tasks like checking if a word is inside another or counting letters.

The code here is written for beginners, so you can read it, run it, and play with it to get comfortable with strings.

Have fun exploring Python text!
